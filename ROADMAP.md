# WETT Roadmap

## v0.1.0 — Foundation ✅

- Modular repository structure
- Safe console menu
- System snapshot
- Network snapshot
- TCP connection mapping
- Security posture snapshot
- HTML/JSON report with SHA-256 manifest
- Logging, documentation, CI, and starter tests

## v0.2.0 — Event Log Triage

- Failed and successful logon summaries
- Account lockout review
- PowerShell operational log collection
- Microsoft Defender event collection
- RDP logon review
- Event ID learning cards
- Configurable time window

## v0.3.0 — Authorized Repair Module

- SFC and DISM health workflow
- Windows Update diagnostics
- DNS cache clearing
- Explorer restart
- Separate admin-only menu
- Confirmation prompts
- Before/after evidence collection
- Rollback and restart guidance

## v0.4.0 — Incident Response Collection

- Scheduled tasks, services, startup entries, drivers, users, groups
- DNS cache and ARP/neighbor cache
- File hashing and analyst notes
- ZIP evidence package
- Chain-of-custody worksheet
- Collection scope and consent record

## v0.5.0 — Enterprise and Homelab

- Remote PowerShell collection
- Active Directory and Entra ID checks
- Sysmon validation
- WSL, Hyper-V, Docker, and VM inventory
- Linux companion scripts

## v1.0.0 — GUI and Portfolio Release

- Signed PowerShell modules
- WinUI/WPF interface
- Role-based views: Help Desk, Network, SOC, Administrator
- Documentation site
- Demo screenshots and video
- Versioned release package
