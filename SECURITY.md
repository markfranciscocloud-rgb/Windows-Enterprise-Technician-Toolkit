# Security Policy

## Supported versions

WETT is currently an early-stage educational project. Only the latest tagged release is supported.

## Reporting a vulnerability

Do not publish exploitable details in a public issue. Use GitHub private vulnerability reporting when enabled, or contact the repository owner privately.

## Sensitive output

Generated logs and reports can contain device names, usernames, IP addresses, serial numbers, installed software, processes, and security configuration. The repository ignores these directories by default. Review every artifact before sharing it.
