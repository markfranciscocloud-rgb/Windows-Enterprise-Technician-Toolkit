# Session 01 — Repository Foundation

## What was built

- Git-ready folder structure
- Main PowerShell menu
- Core logging and error handling
- System diagnostics module
- Network diagnostics module
- Security posture module
- Full HTML/JSON triage report
- SHA-256 integrity manifest
- Pester starter tests
- GitHub Actions quality workflow

## Hands-on tasks

1. Extract the repository to your Windows computer.
2. Open the folder in Visual Studio Code.
3. Run `Start-WETT.bat`.
4. Run options 1, 2, 5, and 6.
5. Generate option 7 and inspect the HTML, JSON, and hash manifest.
6. Open the daily log in `Logs`.
7. Initialize Git and make the first commit.
